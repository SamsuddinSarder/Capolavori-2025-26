#  SmartMarket

### Progettazione e sviluppo di un ecosistema digitale per la gestione intelligente di un supermercato

**Autore:** Sarder Samsuddin
**Corso di studi:** Informatica e Telecomunicazioni
**Progetto TPSIT – Esame di Stato**
**Tecnologie:** HTML5 · CSS3 · JavaScript · PHP 8 · MySQL 8

---

##  Descrizione

SmartMarket è una piattaforma web sviluppata per simulare la gestione completa di un supermercato moderno. Il sistema integra un catalogo prodotti online, la gestione degli utenti, un carrello della spesa e un terminale POS (Point Of Sale) per l'emissione degli scontrini.

L'obiettivo del progetto è applicare le competenze acquisite durante il percorso scolastico nella realizzazione di un'applicazione web completa basata su architettura client-server, database relazionale e API REST.

---

##  Obiettivi

* Realizzare un'applicazione web moderna e responsive.
* Gestire autenticazione e autorizzazione degli utenti.
* Implementare un catalogo prodotti dinamico.
* Sviluppare un sistema di carrello e acquisto.
* Simulare il funzionamento di una cassa POS.
* Progettare un database relazionale normalizzato.
* Applicare principi di sicurezza informatica.
* Utilizzare API REST per la comunicazione client-server.

##  Architettura del Sistema

SmartMarket utilizza una struttura a tre livelli:

| Livello  | Tecnologia            | Funzione           |
| -------- | --------------------- | ------------------ |
| Frontend | HTML, CSS, JavaScript | Interfaccia utente |
| Backend  | PHP 8                 | Logica applicativa |
| Database | MySQL 8               | Persistenza dati   |

La comunicazione tra frontend e backend avviene tramite richieste HTTP e risposte JSON.
